# TypeScript Types Generator

A Yaak plugin that generates TypeScript types from HTTP response JSON.

## Installation

1. Open Yaak → **Settings** → **Plugins**
2. Click "Install from directory"
3. Select this folder

## Usage

1. Make an HTTP request that returns JSON
2. Right-click on the request → **"Generate TypeScript Types"**
3. Types are copied to your clipboard

## Development

```bash
# Install dependencies
npm install

# Build the plugin
npm run build

# Watch mode
npm run dev
```
